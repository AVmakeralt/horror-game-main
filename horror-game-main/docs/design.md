# Design Pillars (Prototype Scope)

## 1) Stalking Threat
The stalker is pressure, not combat. It patrols a graph and switches to alert on player noise.

## 2) Progressive Transformation
Contact with tainted objects grants persistent or timed movement/control modifiers.

## 3) Reactive Environment
Timed events make traversal unreliable: light flickers and corridor doors seal briefly.

## 4) Minimal UI
Only subtle HUD elements are shown to preserve ambiguity and fear.
